# XServiceBuilder
A c# service builder class library for a quicker way to add custom service collections


## AppSetting Options
' "XServiceOptionsCollection": {
    "Options": [
      {
        "Type": "Todo.Options",
        "Data": {
          "ConnectionString": "YOUR_CONNECTION_STING",
          "ANY_OTHER_PROPERTY","ANY_OTHER_PROPERTY_VALUE"
        }
      }
    ]
  }'
